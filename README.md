## 📖 Overview

GH Forensics is a Windows-based Digital Forensics & Incident Response (DFIR) application designed to assist investigators, students and cybersecurity professionals in collecting, analyzing and reporting forensic artifacts from Windows systems.

The application provides an intuitive graphical interface for gathering system evidence, reconstructing user activity, analyzing browser artifacts, inspecting USB history, detecting suspicious files and generating forensic reports.

The project is intended for **defensive security**, **digital forensics**, **incident response** and **educational research**.

# ✨ Features

## 🖥️ System Information

- Computer information
- Windows version
- Logged-in users
- Installed software
- Running processes
- Memory usage

---

## 🌐 Network Analysis

- Active network connections
- Open ports
- DNS cache
- IP information

---

## 📂 Browser Forensics

Supports collecting browser artifacts including:

- Browser History
- Download History
- Cookies metadata
- Recently visited websites

---

## 📁 File System Artifacts

- Recent Files
- Downloads Folder
- Recycle Bin
- Shellbags
- Search History

---

## ⚡ Execution Artifacts

- Prefetch Analysis
- Recently executed applications

---

## 🔌 USB Forensics

- USB Device History
- Device information
- Connection timestamps

---

## 🔍 Malware Scanning

Current capabilities include:

- Suspicious file detection
- SHA256 hashing
- Basic entropy analysis
- File inspection

---

## 📄 Reporting

Generate investigation reports including:

- PDF Reports
- JSON Export
- SHA256 Integrity Verification
